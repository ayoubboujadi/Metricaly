Metricaly: Realtime monitoring service for developers.

Free

Metricaly is completely free, there is no payment required, you just need an email address to register and get immediate access.

Realtime

Monitoring data sent to Metricaly are available immediately either though the online dashboard or through the public API.

1-second granularity

Recent monitoring data are stored with up to 1-second granularity, this allows you to deep dive into the data for more details.

1 Year data retention period

Monitoring data have 12 months retention period at 15 min granularity.

Customizable Dashboards

Build custom dashboards to easily plot an monitor your data.

An easy to use API

Metricaly provides an API that can be used to send metrics data, the API can be used from any programming language or framework.
